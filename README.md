## Descripción
Implementación del artículo "GeMuCo: Generalized Multisensory Correlational Model for Body Schema Learning" de Kawaharazuka et al. (2024).

Este proyecto implementa un modelo de esquema corporal para robots que aprende la relación entre sensores y actuadores usando:
- Red neuronal Encoder-Decoder
- Máscaras para correlación multisensorial
- Parametric Bias para adaptación a cambios
